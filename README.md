# s3-helper

**This repository is no longer maintained.**

It has moved to **[`bucket-helper`](https://github.com/warith-harchaoui/bucket-helper)**.

The Python module is now `bucket_helper`:

```python
import bucket_helper as bh
```

Install:

```bash
pip install --force-reinstall --no-cache-dir \
    git+https://github.com/warith-harchaoui/bucket-helper.git@v0.1.0
```

No functional changes — same surface, same behavior. The rename is purely a
PyPI naming-conflict resolution.

For history, the original `s3-helper` `v0.1.0` tag remains readable on this
archived repository.
